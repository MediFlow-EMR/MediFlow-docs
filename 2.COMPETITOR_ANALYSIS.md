# 경쟁사 분석 (Competitor Analysis)

## 핵심 요약

메디 플로우는 진료 중심의 기존 EMR에 **간호 업무 자동화**를 더해 병원 운영의 완성도를 높인 차세대 솔루션입니다. **AI 기반 인수인계와 통합 환자 뷰**로 불필요한 행정 업무를 줄임으로써, 의료진이 환자 케어에 온전히 집중할 수 있는 환경을 만들고 병원 경영 효율까지 극대화합니다.

## 메디 플로우의 경쟁 우위

### 1. 병원 운영 효율성 극대화

**핵심 가치 제안:**
- 간호사 업무 효율 향상 = 병원 전체 수익성 개선
- 의사는 진료에, 간호사는 환자 케어에 집중 가능
- 신규 간호사 적응 기간 단축으로 교육 비용 절감

### 2. 통합 환자 뷰 (One-View Dashboard)

**기존 EMR 방식:**
```
환자 클릭 → 어디에 뭐가 있는지 모름 → 여러 메뉴 클릭하며 탐색
→ 활력징후 찾음 → I/O 어디있지? → pacs? → 간호기록은?
(정보 위치 파악 어려움, 신규 간호사는 선배에게 물어봐야 함)
```

**메디 플로우 방식:**
```
환자 이름 클릭 → 활력징후·I/O·검사결과·오더조회·투약·간호기록 탭이 한 화면에 정렬되어 표시
→ 원하는 탭만 클릭
(정보 위치가 명확하게 보임, 클릭 한 번으로 즉시 접근)
```

**효과:**
- 신규 간호사도 "어디에 뭐가 있는지" 바로 파악 가능
- 응급 상황 시 필요한 정보에 빠르게 접근
- 의사 회진 시 즉각적인 환자 상태 보고 가능
- 선배 간호사에게 물어보는 시간 절약

### 3. AI 기반 인수인계 자동화

| 항목 | 기존 방식 | 메디 플로우                  |
|-----|---------|-------------------------|
| **인수인계 작성** | 간호사가 수기로 요약 작성 | AI가 당일 간호기록 기반 초안 자동 생성 |
| **정보 일관성** | 간호사마다 작성 스타일 상이 | 표준화된 템플릿 기반             |
| **정보 누락** | 구두 전달 시 누락 위험 | 시스템 기반으로 누락 최소화         |
| **의료진 협업** | 인수인계 시간 때문에 회의 참석 어려움 | 시간 단축으로 병동 회의 참석 가능     |

### 4. 향후 확장 계획 (간호 인력 지원 강화)

메디 플로우는 단순 EMR을 넘어 **간호 인력 관리 플랫폼**으로 발전합니다.

| 기능 | 목적 | 병원 효과 |
|-----|------|---------|
| **수간호사용 자동 근무표 배치** | AI 기반 간호사 스케줄링 | 공정한 근무 배정, 수간호사 업무 경감 |
| **신규 간호사 온보딩 시스템** | EMR 사용법 가이드 챗봇 | 교육 시간 단축, 빠른 현장 투입 |
| **EMR 가이드 챗봇** | "투약 기록 어디 봐요?" 검색 | 업무 중 즉시 문제 해결 |
| **약품 정보 즉시 조회** | 챗봇으로 약품 API 연동 | 투약 오류 감소, 환자 안전성 향상 |
| **시스템 장애 즉시 대응** | EMR 오류 시 수리실 자동 연결 | 다운타임 최소화, 진료 연속성 확보 |

---

**타겟 병원:**
- 100병상 이상 병원 (3교대 근무 운영)
- 간호사 30명 이상 (인수인계 부담 큼)
- 신규 간호사 채용이 잦은 병원 (온보딩 비용 절감 필요)

**핵심 메시지:**
> "의사는 진료에, 간호사는 환자에게 집중할 수 있도록"
> "간호 인력 효율화로 병원 경쟁력을 높입니다"
